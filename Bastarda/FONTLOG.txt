FONTLOG

Textura Libera
==============

This file provides detailed information on the Textura Libera font family.
This information should be distributed along with the Textura Libera fonts
and any derivative works.


Basic Font Information
----------------------

Textura Libera means "Free Textura" and is a free blackletter font family.

Textura Libera development is currently hosted on:
https://github.com/MihailJP/TexturaLibera

Anyone can contribute to Textura Libera by forking the repository and
submitting a pull request on GitHub.
